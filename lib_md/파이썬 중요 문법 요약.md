
## 0. 파이썬 자료형
### 1). 기본 자료형
단 하나의 값만 나타내는 기본 데이터 형태입니다.

- **정수형 (`int`)**: 소수점이 없는 숫자
    
    `x = 10`
    
- **실수형 (`float`)**: 소수점이 있는 숫자
    
    `y = 3.14`
    
- **불리언 (`bool`)**: 참과 거짓을 나타내는 형태
    
    `is_active = True`
    
- **NoneType (`None`)**: "값이 없음"을 나타내는 특별한 형태
    
    `result = None`

### 2). 컨테이너 자료형
- 여러 개의 데이터를 묶어서 관리하는 구조입니다.

| **자료형**           | **예시 코드**         | **특징**                    | **수정 가능 여부 (Mutable)** |
| ----------------- | ----------------- | ------------------------- | ---------------------- |
| **문자열 (`str`)**   | `"Hello"`         | 문자의 순서가 있는 집합             | 불가능 (Immutable)        |
| **리스트 (`list`)**  | `[1, 2, 3]`       | 순서가 있고, 요소를 자유롭게 변경 가능    | **가능 (Mutable)**       |
| **튜플 (`tuple`)**  | `(1, 2, 3)`       | 순서가 있지만, 생성 후 변경 불가       | 불가능 (Immutable)        |
| **세트 (`set`)**    | `{1, 2, 3}`       | 순서가 없고, **중복을 허용하지 않음**   | **가능 (Mutable)**       |
| **딕셔너리 (`dict`)** | `{"name": "Kim"}` | **Key-Value(키-값)** 쌍으로 저장 | **가능 (Mutable)**       |
### 3).이터러블 자료형
- "순회할 수 있는 객체"를 의미합니다. 쉽게 말해 **`for` 문에 집어넣어서 하나씩 요소를 꺼내올 수 있는 모든 데이터 타입**을 이터러블이라고 부릅니다.

- **문자열 (`str`)**: 글자 하나씩 순회
    
    `for char in "Python":` $\rightarrow$ `'P'`, `'y'`, `'t'`...
    
- **리스트 (`list`)**: 요소 하나씩 순회
    
    `for num in [10, 20, 30]:` $\rightarrow$ `10`, `20`, `30`
    
- **튜플 (`tuple`)**, **세트 (`set`)**
    
- **딕셔너리 (`dict`)**: 기본적으로 **Key**를 하나씩 순회
    
    `for key in my_dict:`
    
- **`range()` 객체**: 지정한 범위의 숫자를 순차적으로 생성
#### (1). 이터러블과 같이 쓰는 기능.
- 1.for문 순회
```python
fruits = ["사과", "바나나", "체리"]
for fruit in fruits:
    print(fruit)
```


- 2.멤버십 연산자(in)
```python
names = ["Kim", "Lee", "Park"]
print("Lee" in names)  # True
```

- 3.언패킹 (이터러블 안의 값들을 여러 변수에 나누어 담습니다.)
```python
a, b, c = [1, 2, 3]  # a=1, b=2, c=3
```

- 4.리스트 컴프리헨션
```python
# 문자열(str)도 이터러블이므로 한 글자씩 꺼내어 대문자로 변환!
uppercase_chars = [char.upper() for char in "hello"]
# 결과: ['H', 'E', 'L', 'L', 'O']
```
